# 📖 Project Overview

Buying or selling property requires an understanding of market prices. This project uses Machine Learning to estimate residential property prices in Lahore based on property characteristics such as area, location, number of bedrooms, bathrooms, kitchens, built year, store rooms, and servant quarters.

The application provides a simple web interface where users enter property details and receive an estimated market value generated by a trained Linear Regression model.

# 📊 Dataset

[**Source**]

Lahore House Listings from Zameen.com (2025)
(https://www.kaggle.com/datasets/tahirmehmood0/lahore-house-listings-from-zameen-com-2025)

Approximately **18,000 residential property listings** were used.

# 📈 Model Performance

| Metric | Value |
|---------|-------|
| R² Score | **0.728** |
| Mean Absolute Error (MAE) | **18.86 Million PKR** |
| Root Mean Squared Error (RMSE) | **42.35 Million PKR** |

### Key Findings

- Performs best on common residential properties (approximately **5–20 Marla**).
- Prediction error increases for very large luxury properties.
- Linear Regression tends to underestimate extremely expensive houses.
- A lower-bound safeguard is applied in the deployed API to prevent impossible negative predictions.

# 💻 Installation


```bash
cd lahore-real-estate-price-predictor
```

### Windows



```bash
pip install -r requirements.txt
```

Run the FastAPI server

```bash
uvicorn app.main:app --reload
```

Open your browser

```
http://127.0.0.1:8000/docs
```

# ⚠️ Limitations

- The current implementation uses **Linear Regression** only.
- Performance decreases for luxury properties (>40 Marla).
- Performance decreases for areas with less data (<20 Houses).
- The model supports residential properties in Lahore only.
- Rare feature combinations may produce unrealistic predictions; therefore, the deployed API applies a minimum-price safeguard.
